Welcome to the info page of this chatroom app!

About:

This is a chatroom application created in order to get a job.
The main culprit: https://www.reaktor.com/careers/summer/

Info on usage of this app:

A user of this app is required to log in with an account which can be created here.

A user can create his own chatrooms and invite other users to a chatroom the user belongs to, provided he knows their usernames.

Chatrooms can be left via the "leave" button, however a chatroom will be destroyed if the are no users left in the chatroom

Users may also be added as friends in order to remember your friends' usernames and easily add them to chatrooms (upcoming feature).
